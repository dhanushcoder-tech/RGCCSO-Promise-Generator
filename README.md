# 🤖 RGCCSO AI Prompt Generator

## 📖 Description

RGCCSO AI Prompt Generator is a Python-based AI application that converts simple user ideas into structured prompts using the RGCCSO framework.

The application uses Microsoft's Phi-3 Small Language Model running locally through Ollama.

---

## 🚀 Features

- Generate AI prompts using RGCCSO
- Local AI (Phi-3)
- Offline processing
- Streamlit web interface
- Easy to use

---

## 🛠️ Technologies Used

- Python
- Streamlit
- Ollama
- Phi-3
- Git & GitHub

---

## ▶️ How to Run

```bash
ollama serve
```

```bash
streamlit run app.py
```
